# JavaScript 内置对象/全局 API 规范全量索引（表格版｜含属性简述）

> 口径：以 **ECMA-262 语言内置**为主；附录列出少量“浏览器/Node 通用宿主内置”（非语言规范本体）。
> 说明：下表为“索引级”整理——**覆盖面优先**，每项给一句话简介；新特性支持度取决于运行时版本。

## 9. Date

### 9.1 Date（静态）

| API           | 类型        | 简介                                     |
| ------------- | ----------- | ---------------------------------------- |
| Date()        | function    | 作为函数调用时返回当前时间字符串         |
| new Date(...) | constructor | 构造 Date 实例（多种重载）               |

### 9.2 Date.prototype

按族群分类

**取值（获取本地时间各组件）**

| API/属性            | 返回值类型 | 简介                                                               | 常见坑/备注                |
| ------------------- | ---------- | ------------------------------------------------------------------ | -------------------------- |
| getDate()           | Number     | 返回“月中的天”（1~31）                                             |                            |

| getFullYear()       | Number     | 返回四位年份（如 2025）                                            | 推荐使用，getYear() 已废弃 |
| getHours()          | Number     | 返回小时（0~23）                                                   |                            |
| getMilliseconds()   | Number     | 返回毫秒（0~999）                                                  |                            |
| getMinutes()        | Number     | 返回分钟（0~59）                                                   |                            |
| getMonth()          | Number     | 返回月份（0=1 月 ~ 11=12 月）                                      | 记得 +1 显示给用户         |
| getSeconds()        | Number     | 返回秒（0~59）                                                     |                            |
| getTime()           | Number     | 返回从 1970-01-01 00:00:00 UTC 到现在的毫秒数（时间戳）            | 与 valueOf() 完全等价      |
| getTimezoneOffset() | Number     | 当前时区与 UTC 的偏移分钟数（东区为负，西区为正）例如中国返回 -480 | 夏令时会变化               |

| -------------------- | ---------- | ---------------------------- |
| getUTCDate()         | Number     | UTC 时间中的“天”（1~31）     |
