# struct 结构体

**自定义数据类型，一种数据的组织方式**

- `struct` 实例化必须每个字段都赋值
- 每个字段拥有独立的所有权
- 旧的结构体中的字段必须被新的结构体包含,不能有新结构体没有的字段
- 更新语法必须在新结构体的后面
- 更新语法会转移结构体字段的所有权

## 定义struct

```rust
struct User {
  name: String,
  age: i32,
  email: String,
}
```

## 实例化struct

```rust
let san = User {
  name: String::from("zhangsan"),
  age: 13,
  email: String::from("123@gmail.com")
};
```

## 访问

使用点语法

```rust
let san_name = san.name
```

## 修改

使用点语法

```rust
san.name = "lisan".to_string()
```

## 更新语法

修改使用同一个结构体模板的变量的其中某部分值

```rust
let new_san = User{
email: String::from("123@gmail.com"),
..san
}
```

- san必须是已经通过User实例化的结构体
- 新的new_san变量复用san中除email外的其他字段值
- 更新语法会转移所有权
- 由于struct中的所有权是单独的,实现了Copy的字段在更新语法后依旧可以直接访问

## 元组结构体 tuple structs

```rust
struct Color(i32, i32, i32);
struct Point(i32, i32, i32);

fn main() {
let black = Color(0, 0, 0);
let origin = Point(0, 0, 0);
}
```

> black 和 origin 是不同的类型,哪怕里面的类型都是一样的

## 类单元结构体 unit-like structs

没有任何字段的结构体

```rust
struct AlwaysEqual;

fn main() {
    let subject = AlwaysEqual;
}
```

## 关联函数

- 使用`impl`关键字定义
- 一个结构体可以使用多个`impl`定义结构体关联函数
- 所有在 impl 块中定义的函数/方法被称为 关联函数（associated functions），因为它们与 impl 后面命名的类型相关

**是方法的关联函数**

- 第一个参数是`&self`,指代结构体实例的引用
- &self 其实是 self: &Self 的缩写。这里的 Self（大写）指代类型，而 self（小写）指代当前那个实例

```rust
impl User {
    fn say(&self) {
        println!("我的名字是{}", self.name);
    }
}
```

**不是方法的关联函数**

- 一般用于返回一个结构体新实例的构造函数
- `Self`是一个类型别名，它永远指向当前正在实现（impl）的那个类型
  - `self` (小写)：指代具体的实例（那个“对象”）。
  - `Self` (大写)：指代这个类型本身（那张“图纸”）。

```rust
impl User {
    fn new(name: String,  age: u8, email:  String ) -> Self {
        Self {
            name,
            age,
            email,
        }
    }
}

```

## 调用关联函数

```rust
san.say()
```

## 综合例子

```rust
// 定义结构体
#[derive(Debug)] // 实现Debug trait,可以使用println!宏打印
struct User {
    name: String,
    age: u8,
    email: String,
}

// 定义结构体方法/关联函数
impl User {
    // 非方法,更像是构造函数
    fn new(name: String, age: u8, email: String) -> Self {
        Self { name, age, email }
    }
    // 方法
    fn say(&self) {
        println!("我的名字是{}", self.name);
    }
}

fn main() {
    // 实例化结构体
    let mut san = User {
        name: String::from("zhangSan"),
        age: 18,
        email: "345@gmail.com".to_string(),
    };

    // 更新语法
    let mut si = User {
        email: "1223@gmail.com".to_string(),
        ..san // san.name 和 san.email(String类型)所有权转移给 si
    };

    // san.age(u8,实现了Copy)依然可以访问,但 san.name 和 san.email 已被移走
    dbg!(san.age);

    // 修改san
    san.name = "liSi".to_string(); // 重新赋值,拥有所有权
    dbg!(san.name);

    // 使用say方法
    si.say();

    // 使用new函数,构造新的字段
    let wu = User::new("wu".to_string(), 25, "567@gmail.com".to_string());
    println!("{wu:#?}")
}

```
