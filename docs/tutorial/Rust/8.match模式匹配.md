# match 枚举控制流

- 模式匹配工具,不仅用于枚举
- 穷尽性：编译器要求你必须处理所有可能的情况，否则无法编译通过。这保证了运行时的类型安全
- 表达式性质：`match` 是一个表达式，这意味着它可以返回值，直接用于变量赋值。
- 按序匹配：程序会从上到下检查分支，执行第一个匹配成功的代码块。
- 会移动所有权

## match 用法

```rust
#[derive(Debug)]
enum San {
    eat(String), // 状态吃:吃的什么
    sleep(u8),   // 状态睡:睡了多久
    run(i32),    // 跑了多远
    jump(i32),   // 跳了多远
}

fn main() {
    // 定义san的状态为跳,跳了2米
    let san = San::jump(2);
    // match匹配san的状态
    match san {
        San::eat(foot) => {
            println!("吃的是: {foot:?}")
        }
        San::sleep(hours) => {
            println!("睡了{hours:?}小时")
        }
        other => {
            println!("除了吃和睡的其他情况: {other:?}")
        } // 匹配其他可能,使用里面的值
        _ => {
            println!("除了吃和睡的其他情况")
        } // 匹配其他可能,不在乎里面的值
      // 注意: 最后的占位符匹配永远不会执行,根据按序匹配other已经处理了
    }
}
```

## if let /if let else

- match 语法糖
- 只关心一种情况,也可以更精确的关心这种情况的某些值,其他情况都不关心
- 本质是: 如果中了就干点啥

```rust
// 1. 如果san是跳了,我就干点啥,我不关心跳了多少米,其他情况我也不关心
if let San::jump(_) = san {
    println!("就是要跳跃")
} else {
    println!("其他情况: {san:?}")
}
// 2. 如果san是跳了2米,我就干点啥,其他情况我不关心
if let San::jump(2) = san {
    println!("就是要跳跃2米")
} else {
    println!("其他情况: {san:?}")
}
// 3. 如果san是跳了,获取跳了多少米,我就干点啥,其他情况我也不关心
if let San::jump(jump_distance) = san {
    println!("就是要跳跃,跳了{jump_distance}米")
} else {
    println!("其他情况: {san:?}")
}
```

## let-else

- 守卫模式/解构失败退出
- 如果不匹配，我就直接 `return`退出
- 必须包含 `else` 块。
- `else` 块里必须中止当前流程（用 `return、break、panic`! 等）
- 如果匹配成功，变量会直接绑定到当前作用域，后面能直接用
- 本质是: 如果不中我就不干了,直接退出(符合条件才让程序继续执行,否则直接退出)

```rust
let San::jump(2) = san else {
        panic!("不是跳了2米") // 程序崩溃
    };
```

## 额外补充

### 多模式匹配 `|`

使用 `|` 在同一分支中匹配多个模式：

```rust
let x = 2;
match x {
    1 | 2 => println!("1 或 2"),
    3 => println!("3"),
    _ => println!("其他"),
}
```

### 范围匹配 `..=`

使用 `..=` 匹配闭区间范围（只支持数字和 `char`）：

```rust
let x = 5;
match x {
    1..=5 => println!("1 到 5"),
    6..=10 => println!("6 到 10"),
    _ => println!("其他"),
}
```

### `@` 绑定

`@` 让你在**测试一个值的同时将其绑定到变量**，可以同时用值做匹配判断又保留它：

```rust
let x = 5;
match x {
    n @ 1..=10 => println!("1到10之间的值: {n}"),
    n @ 11..=20 => println!("11到20之间的值: {n}"),
    _ => println!("不在范围内"),
}
```

### 匹配守卫（match guard）

在分支后加 `if` 条件进行附加过滤，编译器**不会将守卫纳入穷举检查**：

```rust
let num = Some(4);
match num {
    Some(x) if x % 2 == 0 => println!("偶数: {x}"),
    Some(x) => println!("奇数: {x}"),
    None => println!("无值"),
}
```

### 解构 struct / tuple

match 中可以直接解构复合类型：

```rust
struct Point { x: i32, y: i32 }
let p = Point { x: 0, y: 5 };

match p {
    Point { x: 0, y } => println!("在 y 轴上，y = {y}"),
    Point { x, y: 0 } => println!("在 x 轴上，x = {x}"),
    Point { x, y }    => println!("({x}, {y})"),
}
```

```rust
// 元组解构
let pair = (true, 42);
match pair {
    (true, n) => println!("为 true，值为 {n}"),
    (false, _) => println!("为 false"),
}
```

### `..` 忽略剩余字段

```rust
struct Point3D { x: i32, y: i32, z: i32 }
let p = Point3D { x: 1, y: 2, z: 3 };

match p {
    Point3D { x, .. } => println!("只关心 x = {x}"),
}
```
