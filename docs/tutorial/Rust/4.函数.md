## 函数定义

```rust
fn add_num(x: i32, y: i32) -> i32 {
    x + y // 隐式返回,一定不能加分号,会变成语句
}
```

```rust
// if 也是表达式,也能直接返回值
fn max(a: i32, b: i32) -> i32 {
    if a > b {
        a
    } else {
        b
    }
}
```

- `main` 函数是入口点。
- 使用 `fn` 关键字来声明新函数。
- 函数默认返回 `()`
- snake case 规范风格: 所有字母都是小写并使用下划线分隔单词。
- 在函数签名中，必须 **声明每个参数的类型 和 返回值的类型**
- 函数返回值在箭头`->`后声明它的类型
- 函数的返回值等同于函数体最后一个表达式的值
- 使用 `return` 关键字和指定值，可从函数中提前返回

## 语句和表达式

- 语句（Statements）是执行一些操作但不返回值的指令,语句的值是()。
- 表达式（Expressions）计算并产生一个值。

> rust 中不用分号默认代表是返回一个表达式,也就是默认调用 return

**语句包括：**

1. 变量声明
1. 函数定义
1. 表达式 + 分号

**表达式包括：**

1. 数值
1. 运算
1. 函数调用
1. 块
1. if
1. match
1. loop

甚至：

```rust
{
let x = 5;
x + 1
}
```

这是一个块表达式（block expression）,它的值是最后一行,返回6。
