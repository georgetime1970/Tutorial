# 向量（vector）

**本质是可变数组**

- vector 只能储存相同类型的值
- 和数组一样,是整体所有权

## 新建 vector

```rust
// Vec::new 函数
let v: Vec<i32> = Vec::new();   // 空数组 []

// vec! 宏
let v = vec![1, 2, 3];
```

## 增加元素

- 使用push方法

```rust
let v = vec![1, 2, 3, 4, 5];
v.push(5);
```

## 读取元素

1. 使用 `[]` 读取元素
2. 使用`get()`方法读取元素

> 当超过最大索引,`[]`会panic,`get()`会返回None

```rust
let v = vec![1, 2, 3, 4, 5];

// 使用[2]索引
let third: &i32 = &v[2];
println!("The third element is {third}");

// 使用get方法,得到的是 Option枚举
// get方法不会移动所有权
let third: Option<&i32> = v.get(2);
match third {
    Some(third) => println!("The third element is {third}"),
    None => println!("There is no third element."),
}
```

## 遍历 vector 中的元素

```rust
let v = vec![100, 32, 57];
for i in &v {
    println!("{i}");
}
```

```rust
let mut v = vec![100, 32, 57];
for i in &mut v {
    *i += 50; // 修改vector的值
}
```

## 使用枚举来储存多种类型

- vector 只能储存相同类型的值
- 枚举本身就是类型

```rust
enum SpreadsheetCell {
    Int(i32),
    Float(f64),
    Text(String),
}

let row = vec![
    SpreadsheetCell::Int(3),
    SpreadsheetCell::Text(String::from("blue")),
    SpreadsheetCell::Float(10.12),
];
```
