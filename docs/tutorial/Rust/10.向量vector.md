# 向量（vector）

**本质是可变数组**

- vector 只能储存相同类型的值
- 和数组一样,是整体所有权

## 新建 vector

```rust
// Vec::new 函数
let v: Vec<i32> = Vec::new();   // 空数组 []

// vec! 宏
let v = vec![1, 2, 3];
```

## 增加元素

- 使用push方法

```rust
let mut v = vec![1, 2, 3, 4, 5]; // 必须声明 mut
v.push(5);
```

## 读取元素

1. 使用 `[]` 读取元素
2. 使用`get()`方法读取元素

> 当超过最大索引,`[]`会panic,`get()`会返回None

```rust
let v = vec![1, 2, 3, 4, 5];

// 使用[2]索引
let third: &i32 = &v[2];
println!("The third element is {third}");

// 使用get方法,得到的是 Option枚举
// get方法不会移动所有权
let third: Option<&i32> = v.get(2);
match third {
    Some(third) => println!("The third element is {third}"),
    None => println!("There is no third element."),
}
```

## 遍历 vector 中的元素

```rust
let v = vec![100, 32, 57];
for i in &v {
    println!("{i}");
}
```

```rust
let mut v = vec![100, 32, 57];
for i in &mut v {
    *i += 50; // 修改vector的值
}
```

## 使用枚举来储存多种类型

- vector 只能储存相同类型的值
- 枚举本身就是类型

```rust
enum SpreadsheetCell {
    Int(i32),
    Float(f64),
    Text(String),
}

let row = vec![
    SpreadsheetCell::Int(3),
    SpreadsheetCell::Text(String::from("blue")),
    SpreadsheetCell::Float(10.12),
];
```

## 额外补充

### 常用增删方法

```rust
let mut v = vec![3, 1, 4, 1, 5, 9];

v.pop();              // 移除并返回最后一个元素 → Option<T>
v.remove(0);          // 移除索引 0 的元素（后续元素左移），返回该元素
v.insert(1, 10);      // 在索引 1 处插入 10（原有元素右移）
v.drain(0..2);        // 移除索引 0..2 的元素，返回被移除元素的迭代器
v.retain(|x| *x > 2); // 只保留满足条件的元素（原地过滤）
v.dedup();            // 去除相邻重复元素（配合 sort 可实现完全去重）
v.clear();            // 清空向量
```

### 常用查询方法

```rust
let v = vec![1, 2, 3, 4, 5];

v.len();              // 元素个数
v.is_empty();         // 是否为空
v.contains(&3);       // 是否包含某值 → bool
v.iter().position(|x| *x == 3); // 找到首个满足条件的索引 → Option<usize>
```

### 排序

```rust
let mut v = vec![3, 1, 4, 1, 5];
v.sort();                         // 对实现了 Ord 的类型排序（整数、字符串等）
v.sort_by(|a, b| b.cmp(a));       // 自定义排序（这里是降序）
v.sort_by_key(|k| std::cmp::Reverse(*k)); // 按 key 排序

let mut floats = vec![3.1f64, 1.4, 2.7];
floats.sort_by(|a, b| a.partial_cmp(b).unwrap()); // f64 没实现 Ord，需用 partial_cmp
```

### 合并与扩展

```rust
let mut a = vec![1, 2, 3];
let b = vec![4, 5, 6];
a.extend(b.iter());      // 追加 b 的元素到 a（b 依然有效，只借用）
a.extend([7, 8, 9]);     // 也可以直接追加数组
```

### 容量与预分配

```rust
// Vec::with_capacity 预先分配空间，避免频繁扩容，适合已知元素数量时
let mut v: Vec<i32> = Vec::with_capacity(100);
v.push(1);
println!("len={}, capacity={}", v.len(), v.capacity()); // len=1, capacity=100
```
