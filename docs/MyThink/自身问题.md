# 遇到的问题

## 意志和身体的问题

意志消沉，需要成功学、励志鸡汤

身体素质降低，需要加强锻炼

这两项对学习工作是非常重要的，意志消沉会严重影响学习和工作，身体素质降低会增加患病风险同时加重意志消沉，形成恶性循环。

身体是革命的本钱，是一切成功的基本

## 学习和实践的问题

由于意志消沉和身体素质降低，学习和工作将受到严重影响，一旦停止学习，就会陷入信息茧房中停滞不前，一样会陷入消沉的恶性循环中，更无从谈起将学习用于实践。

无论何时不要停止学习的步伐
